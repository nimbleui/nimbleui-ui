## API

### Select Props
| 名称 | 类型 | 默认值 | 说明 | 版本 |
| --- | --- | --- | --- | --- |
| name | `number\|string` | - | 唯一标识，给form收集数据 | - |
| options | `Array` | - | 配置项 | - |
| disabled | `boolean` | - | 禁用 | - |
| modelValue | `number\|string` | - | 绑定的值 | - |
