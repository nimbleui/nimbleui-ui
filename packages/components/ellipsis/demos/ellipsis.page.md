# 文本省略 Ellipsis
文本溢出隐藏

```demo
basic.vue
max.vue
custom.vue
```
## API

### Ellipsis Props
| 名称 | 类型 | 默认值 | 说明 | 版本 |
| --- | --- | --- | --- | --- |
| lineClamp | `number` | `0` | 最大行数 | |
| maxWidth | `string \| number` | - | 最大宽度 | |

### Ellipsis Slots
| 名称 | 参数 | 说明 |
| --- | --- | --- |
| default | - | 文本省略的内容 |
| tooltip | - | 提示的的内容 |
