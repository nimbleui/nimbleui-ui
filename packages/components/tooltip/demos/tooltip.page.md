# 文字提示 Tooltip
简单的文字提示气泡框。

```demo
basic.vue
location.vue
```
## API

### Tooltip Props
| 名称 | 类型 | 默认值 | 说明 | 版本 |
| --- | --- | --- | --- | --- |
| lineClamp | `number` | `0` | 最大行数 | |
| maxWidth | `string \| number` | - | 最大宽度 | |

### Tooltip Slots
| 名称 | 参数 | 说明 |
| --- | --- | --- |
| default | - | 文本省略的内容 |
| tooltip | - | 提示的的内容 |
