# 分割线 Divider
区隔内容的分割线。

```demo
basic.vue
title.vue
vertical.vue
```
## API

### Divider Props
| 名称 | 类型 | 默认值 | 说明 | 版本 |
| --- | --- | --- | --- | --- |
| direction | `"left"` \| `"center"` \| `"right"` | `'center'` | 标题的位置 | |
| dashed | `boolean` | `false` | 是否使用虚线分割 | |
| vertical | `boolean` | `false` | 是否垂直分隔 | |
