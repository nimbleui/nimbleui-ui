## API

### Number Props
| 名称 | 类型 | 默认值 | 说明 | 版本 |
| --- | --- | --- | --- | --- |
| count | `string \| number` | - | 标记数量 | - |
| max | `string \| number` | - | 展示封顶的数字值 | - |
| numberClass | `string \| Array \| Object` | - | 数字的类名 | - |
| numberStyle | `string \| Array \| Object` | - | 数字的样式 | - |
| gap | `string \| number ` | - | 数字间的间隙 | - |
